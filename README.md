# Mineral Collection Site

Welcome to the Mineral Collection Site project! This application allows users to create a collection of minerals by adding details and uploading photos.

## Project Structure

```
mineral-collection-site
├── src
│   ├── index.html        # Main HTML document
│   ├── styles.css       # Styles for the application
│   ├── app.js           # Main JavaScript file
│   ├── components
│   │   ├── mineral-form.js  # Form for inputting mineral details
│   │   └── mineral-list.js   # Displays the list of minerals
│   └── assets
│       └── images       # Directory for mineral images
├── package.json         # npm configuration file
└── README.md            # Project documentation
```

## Features

- Add new minerals with details such as name, description, and photo.
- View a list of all minerals added to the collection.
- User-friendly interface for managing mineral entries.

## Setup Instructions

1. Clone the repository:
   ```
   git clone <repository-url>
   ```

2. Navigate to the project directory:
   ```
   cd mineral-collection-site
   ```

3. Install the dependencies:
   ```
   npm install
   ```

4. Start the server:
   ```
   npm start
   ```

5. Open `http://localhost:8080` in your web browser to view the application.

## Usage Guidelines

- Use the form to input details about each mineral.
- Upload photos to visually represent your minerals.
- The mineral list will automatically update as new entries are added.
